# Maquina De Cafe 
Implementando Test Driven Development

## Preparacion / Uso
Desde la terminal, dentro del directorio del proyecto:
```bash
$ npm install
```
 Ejecutar pruebas
 ```bash
 $ npm test
```
 Ejecutar aplicacion
 ```bash
 $ npm start
```
## Tecnologias
- NodeJS
- Typescrypt
- Jest
- inquirer

## Patrones/Metodologias/Modelos
- POO
- TDD
