# Maquina De Cafe 
Implementando Test Driven Development

## preparacion
Desde la terminal, dentro del directorio del proyecto:
 - $ npm install

 Ejecutar pruebas
 - $ npm test

 Ejecutar aplicacion
 - $ npm start

## Tecnologias
- NodeJS
- Typescrypt
- Jest
- inquirer

## Patrones/Metodologias/Modelos
- POO
- TDD
